Example usage of:

- https://developers.cloudflare.com/pages/framework-guides/deploy-a-hono-site/
- https://hono.dev/getting-started/cloudflare-workers 


1. Create a free account in CloudFlare
2. Install Node.js
3. Install pnpm
4. How to run:
```
pnpm i
pnpm dev
pnpm build
pnpm deploy
```

`pnpm deploy` will ask to login/signup for CloudFlare
